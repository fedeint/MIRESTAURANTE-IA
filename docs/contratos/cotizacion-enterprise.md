## RESUMEN EJECUTIVO

MiRestconIA es un sistema integral de gestión para restaurantes con inteligencia artificial integrada. Gestiona mesas, pedidos, cocina, caja, inventario, facturación electrónica SUNAT, y envío de comprobantes por WhatsApp. Funciona en la nube y localmente cuando no hay internet.

## QUÉ INCLUYE

### Software (licencia perpetua)

| Módulo | Descripción |
|--------|-------------|
| **Mesas** | Gestión visual de mesas, zonas, estados, asignación de meseros |
| **Pedidos** | Catálogo visual de productos, toma de pedidos desde tablet/celular |
| **Cocina** | Pantalla de cocina en tiempo real, estados de preparación |
| **Caja** | Apertura/cierre, pagos mixtos (efectivo, tarjeta, Yape, Plin) |
| **Facturación SUNAT** | Boletas y facturas electrónicas vía NubeFact (OSE autorizado) |
| **WhatsApp** | Envío de comprobantes al cliente por WhatsApp |
| **Almacén** | Control de inventario, lotes, alertas de stock bajo, mermas |
| **Recetas** | Vinculación ingredientes-platos, disponibilidad automática |
| **Productos** | Catálogo con fotos, precios, categorías, variantes |
| **Reportes PDF** | Resumen del día + lista de compras para mañana |
| **DalIA (IA)** | Asistente inteligente: tareas, insights, alertas (2M tokens/año) |
| **Clientes** | Base de datos de clientes |
| **Administración** | P&L, gastos, planilla |
| **Delivery** | Gestión de pedidos para llevar/delivery |
| **Reservas** | Sistema de reservas |
| **Usuarios** | Gestión de staff con 5 roles diferenciados |
| **Modo Local** | Funciona sin internet (cortes de luz) |
| **Backups** | Respaldos automáticos de toda la información |
| **Soporte** | Sistema de tickets integrado |
